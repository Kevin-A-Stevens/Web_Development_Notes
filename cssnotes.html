Styling text

color: value = changes text color

font-size: value = size of text

font-family: values = specify type of font

text-transform: uppercase = change text (uppercase, lowercase, capitalize)

font-style: value = change style of text like to italic

line-height: 1.5 = change line height or space between lines

text-align: center = aligns text to center, end, start

font-weight: bold = change weight of font, can also use numbers

list-style: none = used to remove bullet points from lists

Combining Selectors

h1, h2, h3 {} = select h1, h2, and h3 elements

header p {} = select all p elements inside a header selector. Descendants

article header p {} = select all p elements inside a header element that is inside an article element

h3 + p::first-line {} = + is the adjacent sibling selector

Styling links

a:link {} = style a elements that only have a href attribute

text-decoration: none; = remove the underline on links

a:visited {} = assign css to a link that has been clicked on before

a:hover {} = adds css to a link when cursor hovers over it

a:active {} = adds css to a link that is active

Layouts

Flexbox

Flex container: display, gap, justify-content, align-items, flex-direction, flex-wrap, align-content

Flex items: align-self, flex-grow, flex-shrink, flex-basis(width), flex (shorthand for prev 3), order(all elements have an order of 0 unless specified with order)

Set display: flex on the container

flex is shorthand for flex-grow, flex-shrink, and flex-basis

CSS Grid

Grid container: grid-template-rows, grid-template-columns, row-gap (gap), column-gap (gap), justify-items, align-items, justify-content, align-content

Grid items: grid-column, grid-row, justify-self, align-self

box-sizing: border-box; = padding and border are included in the width and height:

use fr units when creating your rows and columns

Placing grid items

.grid-item{ grid-column: 2 / 3;} = start at column 2 and end at column 3

.grid-item:{ grid-row: 1 / 2;} = start at row 1 and end at row 2

.grid-item:{ grid-row: 1 / span 3;} = start at row 1 and span 3 cells

.grid-item:{ grid-row: 1 / -1;} = start at row 1 and go to end

Aligning tracks in a container with CSS grid

justify-content: center = align horizontally

align-content: center = align vertically
