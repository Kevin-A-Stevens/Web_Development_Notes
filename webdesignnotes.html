Web Design Notes

Seven steps to building a web site

  1. Define
    1. Who is the website for
    2. What is the website for
    3. What is the main goal/purpose
    4. Who is the target audience
    5. Be specific. These can come from the client
  2. Plan
    1. Gathering the content: copy(text), images, videos, etc
    2. Content is usually provided by client
    3. find free images
    4. Charge if not easy
    5. Plan out the site map for big sites
    6. Based on content, plan what sections the site needs
    7. Define the website personality
  3. Sketch
    1. Think about what components and layout patterns you need
    2. Get ideas and sketch with pen and paper or by using software such as Figma
    3. This is an iterative process
    4. Experiment with different ideas, components, and layouts
    5. Don't need to sketch everything and don't make it perfect
    6. Content should guide the design
  4. Design and Build
    1. Use decisions, content, and sketches to design using HTML and CSS
    2. Design the actual visual styles you decided on and website personality or personalities
    3. Use clients branding if it exists (colors, typography, icons, etc)
  5. Test and Optimize
    1. Make sure website works in all major browsers (Chrome, Safari, Edge, Firefox)
    2. Test on actual mobile devices - not just dev tools
    3. Optimize all images in dimensions and file size
    4. Fix accessability problems (color contrast issues
    5. Run the Lighthouse performance test in Chrome Dev Tool and try to fix reported issues
    6. Think about Search Engine Optimization (SEO)
  6. Launch
    1. Share it with the world
    2. Netlify has a free plan but there are others
    3. Choose and buy a great domain name, one that represents the brand well, memorable, easy to write
  7. Maintain and Update
    1. Keep the website content updated over time
    2. Create a monthly maintenance contract with your client
    3. Install analytics software such Google Analytics or Fathom
    4. A blog is updated regularly is a good way to keep users coming back, and is also good for SEO

Website Personalities - Choose one

  1. Serious/Elegant = thin serif typefaces, golden or pastel colors, big high quality images
  2. Minimalist/Simple = small to medium sans-serif, black text lines and few images/icons
  3. Plain/Neutral = neutral small typefaces, very structured layout. Big corporations.
  4. Bold/Confident = Big and bold typography paired with big/bright colored blocks.
  5. Calm/Peaceful = care. Calming pastel colors. soft serif headings matching images/illustrations.
  6. Startup/Upbeat = startup, featured medium sized sans-serif typefaces. light-grey text/backgrounds
  7. Playful/Fun = colorful/rounds designs. Hand-drawn/creative elements, animations, and fun language

Rules of Website Design

Rule #1

  1. Typography - Choose one or two, not more
  2. Use https://fonts.google.com/ or https://type-scale.com/
  3. Use only good/popular typefaces. Limit # of typefaces to 2
  4. Sans-serif: inter, open sans, roboto, montserrat, work sans lato
  5. Sans: merriweather, aleo, playfair display, cormorant, cardo, lora
  6. Choose font-size, limit choices, use type-scale
  7. normal text: 16px to 32px
  8. long text: 20px or bigger such as a long blog post
  9. headlines: big like 50px+ and bold 600+. Depends on personality
  10. any text: no font weight under 400 (regular)
  11. Use less than 75 characters per line
  12. normal sized text, use line height between 1.5 and 2
  13. big text go below 1.5 line height
  14. decrease letter spacing in headlines if it looks unnatural
  15. Experiment with all caps for short titles. Small and bold increases letter spacing
  16. Don't center long text blocks. Small blocks are fine
